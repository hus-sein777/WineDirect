# Wine Store — Java OOP Project

A wine e-commerce system built in Java for my first-year Computer Science coursework at the University of Salford.

## What it does
- Customers can browse as visitors or register as buyers (with age verification, 18+)
- Buyer accounts get a generated customer ID and can log in
- Shopping basket: add wine cases, view basket, remove items by position, reference, or price limit
- Checkout with wallet balance checking and an automatic 10% discount on orders of 5+ cases

## Structure
- `Browser.java` — the customer: account details, wallet, and basket management
- `Website.java` — the shop: login, registration, and checkout logic
- `WineCase.java` — the product: reference, description, bottles, price

## Built with
Java, developed in BlueJ as part of my BSc Computer Science with Cyber Security.
